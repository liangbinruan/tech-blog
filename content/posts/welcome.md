## 欢迎！🎉

这是您的个人技术博客，使用 Hugo + GitHub Pages 搭建。

### 主要功能

- 📝 **arXiv 论文追踪**: 自动抓取最新 AI/ML 论文
- 📱 **WeChat 文章整理**: 整合优质技术文章
- 🚀 **零运维部署**: GitHub Actions 自动发布
- 📊 **飞书通知**: 新文章自动推送通知

### 下一步

1. 修改 `hugo.toml` 中的作者信息
2. 创建您的第一篇原创教程
3. 配置 GitHub Actions 自动部署
4. 集成飞书通知

### 文章分类

- `arxiv/` - arXiv 论文自动抓取
- `wechat/` - WeChat 文章精选
- `tutorials/` - 原创教程
- `insights/` - 技术洞察

祝您写作愉快！✨
